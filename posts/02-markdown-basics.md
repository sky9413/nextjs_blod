# Markdown 基礎教學

Markdown 是一種輕量級標記語言，讓您可以使用純文字編輯器撰寫格式化的文件。

## 標題

使用 `#` 符號建立標題：

```markdown
# H1 標題
## H2 標題
### H3 標題
```

## 文字格式

- **粗體**：使用 `**文字**` 或 `__文字__`
- *斜體*：使用 `*文字*` 或 `_文字_`
- ~~刪除線~~：使用 `~~文字~~`

## 列表

### 無序列表

使用 `-`、`*` 或 `+`：

```markdown
- 項目 1
- 項目 2
- 項目 3
```

### 有序列表

使用數字加點：

```markdown
1. 第一項
2. 第二項
3. 第三項
```

## 連結和圖片

### 連結

```markdown
[連結文字](https://example.com)
```

範例：[Google](https://google.com)

### 圖片

```markdown
![替代文字](圖片URL)
```

## 程式碼

### 行內程式碼

使用反引號：`const hello = "world";`

### 程式碼區塊

使用三個反引號：

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## 引用

使用 `>` 符號：

> 這是一段引用文字。
> 可以跨越多行。

## 表格

```markdown
| 欄位 1 | 欄位 2 | 欄位 3 |
|--------|--------|--------|
| 內容 1 | 內容 2 | 內容 3 |
| 內容 4 | 內容 5 | 內容 6 |
```

| 語法 | 說明 | 範例 |
|------|------|------|
| `**粗體**` | 粗體文字 | **粗體** |
| `*斜體*` | 斜體文字 | *斜體* |
| `` `code` `` | 行內程式碼 | `code` |

## 總結

Markdown 語法簡單易學，非常適合撰寫 Blog 文章。掌握這些基本語法，您就可以開始創作了！
